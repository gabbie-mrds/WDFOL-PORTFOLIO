This is the very first website that I created and designed. And the whole concept was made by me and it showcases what a coffee shop offers.


With this website I did not use any JavaScript or CSS Frameworks, it was only pure HTML and CSS. The concept behind this website is a local coffee shop that is located in La Union. From the colors, logo, design, and up to contents, I did all of them. Backstory, I remember the time that I created this I actually struggled a lot especially when it comes to CSS, that's why using CSS Frameworks is such a great help.
